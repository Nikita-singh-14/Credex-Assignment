# User Interviews (Mock Data)

These interviews represent realistic target customers for the AI Spend Audit tool.

## Startup A: 'Acme SaaS'
**Profile:** Seed-stage B2B SaaS startup (7 employees).
**Current AI Stack:** 
- 5 Cursor Seats
- 7 ChatGPT Seats
- Open API Usage (~$300/month)
**Total Monthly AI Spend:** $540
**Pain Points:** "We don't realize how much we're spending across 3 different company credit cards and 5 individual employee cards."
**Value Prop:** Consolidating billing and saving ~20% automatically is a no-brainer.

## Startup B: 'Beta Build'
**Profile:** Bootstrap solo founder.
**Current AI Stack:**
- 1 Cursor Seat
- 1 Claude Seat
- Negligible API usage.
**Total Monthly AI Spend:** $40
**Pain Points:** "I'm optimizing for runway, but $40/month is already pretty low."
**Value Prop:** The tool still proves they can save $8/month, which pays for a coffee, but they might not be the ideal high-LTV lead.

## Startup C: 'Charlie GenAI'
**Profile:** Series A AI wrapper startup (15 employees).
**Current AI Stack:**
- 10 Cursor Seats
- OpenAI API Usage (~$4,500/month)
**Total Monthly AI Spend:** $4,700
**Pain Points:** "API costs are eating our margins. If we can get bulk discounts on OpenAI through a partner, we will switch today."
**Value Prop:** Huge LTV. The prospect of saving $675/month on APIs alone is highly attractive.
